Integração com Google Sheets

Configuração única

Para salvar automaticamente no Google Sheets, siga os passos abaixo:

1. Abra o Google Sheets, crie uma nova planilha e copie o ID da URL:
sheets.google.com/d/[ID AQUI]/edit

2. Abra o Google Apps Script, cole o código abaixo e faça o deploy como Web App (acesso: "Qualquer pessoa").

function doGet(e) { var sheet = SpreadsheetApp.openById("SEU_ID_AQUI").getActiveSheet(); if (sheet.getLastRow() === 0) { sheet.appendRow(["Data","Cliente","Produto","Qtd","Valor"]); } var p = e.parameter; sheet.appendRow([p.data, p.cliente, p.produto, p.qtd, p.valor]); return ContentService.createTextOutput("ok"); }

Importar dados do Sheets

Sincroniza as vendas do Google Sheets para este dispositivo. Use ao trocar de aparelho ou após registrar vendas em outro dispositivo.

Alterar senha de acesso

A senha padrão é vendas3d. Troque por uma senha pessoal.

Dados locais

Todos os dados ficam salvos no seu navegador (localStorage) como backup. A integração com Sheets envia cada venda em tempo real.
